# Video Game Sales Forecasting

## Overview

This project analyzes historical video game sales data to identify patterns that may help predict whether a game will succeed commercially. The analysis focuses on platform trends, genre performance, regional sales differences, review scores, and ESRB ratings.

This project was completed as part of the TripleTen Data Science Bootcamp.

## Dataset

The dataset includes:

- Game titles
- Platforms
- Genres
- Release years
- Regional sales data
- Critic scores
- User scores
- ESRB ratings

Data through 2016 was used to analyze trends and build recommendations for 2017 advertising planning.

## Tools

- Python
- pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

## Project Tasks

- Cleaned and prepared the data
- Handled missing values and `tbd` entries
- Calculated total global sales
- Analyzed platform sales trends
- Compared regional market preferences
- Evaluated the relationship between reviews and sales
- Tested statistical hypotheses using t-tests
- Summarized findings and business recommendations

## Key Findings

- Recent platform data was more useful for forecasting than older historical data.
- PS4 and Xbox One showed strong market performance heading into 2017.
- Regional preferences differed significantly between NA, EU, and JP markets.
- Critic scores had a stronger relationship with sales than user scores.
- Shooter and Action games performed strongly across multiple markets.
